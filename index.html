Latest AI Project

AI Teacher

The Java AI Teacher project is designed to assist students in learning Java programming effectively. It utilizes SQLite3 to store student progress, track assignments, and manage learning materials. Tkinter provides an intuitive interface for interacting with the AI, while TensorFlow powers the machine learning algorithms that adapt the teaching style based on the student's performance. The AI offers personalized feedback, suggests learning paths, and helps with problem-solving, ensuring a tailored and engaging educational experience for each user.

Latest Android Project

Amount allotment

I developed an amount allotment application using Kotlin and Jetpack for a seamless user experience, with Firebase serving as the backend to store and retrieve data. The app allows users to enter an amount (e.g., 200 rupees) and automatically calculates how much each person has to pay. Users input the number of people sharing the amount, and the app dynamically displays the individual payment share. Firebase ensures real-time data synchronization, providing an efficient and reliable solution for managing group payments and allotments.
